Shutdown

Kann das System herunterfahren, neustarten oder abmelden.

Shutdown, Code

Parameter

Code

Eine Kombination von den unten genannten Codes.

Bemerkungen

Der Code ist eine Kombination von folgenden Werten:

Abmelden 0
Herunterfahren 1
Neustarten 2
Erzwingen 4
Ausschalten 8
Anhalten/Ruhezustand Siehe DllCall-Beispiel im untersten Abschnitt.
Monitor ausschalten Siehe PostMessage-Beispiele.

Addiere die gewünschten Werte. Zum Beispiel ist zum Herunterfahren und Ausschalten des Systems eine 9 notwendig (Herunterfahren + Ausschalten = 1 + 8 = 9).

Der "Erzwingen"-Wert (4) zwingt alle offenen Programme sich zu schließen. Diese Methode sollte nur im Notfall verwendet werden, weil offene Programme dadurch Daten verlieren könnten.

Der "Ausschalten"-Wert fährt das System herunter und schaltet den Strom aus.

Desweiteren kann mithilfe von OnExit ein Script dazu gebracht werden, zu erkennen, wann das System herunterfährt oder der Benutzer sich abmeldet.

Siehe auch

Run, ExitApp, OnExit

Beispiel

; Erzwingt einen Neustart (Neustart + Erzwingen = 2 + 4 = 6):
Shutdown, 6

; Ruft die API-Funktion "SetSuspendState" von Windows auf, um das System zu sperren oder in den Ruhemodus zu versetzen.
; Parameter #1: Übergebe 1 anstatt 0, um den Ruhemodus zu aktivieren, anstatt anzuhalten.
; Parameter #2: Übergebe 1 anstatt 0, um das System ohne Nachfrage sofort anzuhalten.
; Parameter #3: Übergebe 1 anstatt 0, um alle Aufwachereignisse zu deaktivieren.
DllCall("PowrProf\SetSuspendState", "int", 0, "int", 0, "int", 0)
