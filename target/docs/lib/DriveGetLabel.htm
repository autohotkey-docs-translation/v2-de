DriveGetLabel

Gibt die Volumebezeichnung eines bestimmten Laufwerks zurück.

Bezeichnung := DriveGetLabel(Laufwerk)

Parameter

Laufwerk

Typ: Zeichenkette

Der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und optional einem umgekehrten Schrägstrich, oder ein UNC-Name wie "\server1\share1".

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt die Volumebezeichnung von Laufwerk zurück.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.
