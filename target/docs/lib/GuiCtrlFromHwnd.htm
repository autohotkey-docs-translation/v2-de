GuiCtrlFromHwnd

Ruft das GuiControl-Objekt eines GUI-Steuerelements über dessen Fensterhandle ab.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

Parameter

Hwnd

Typ: Integer

Das Fensterhandle (HWND) eines GUI-Steuerelements oder eines Unterfensters eines solchen Steuerelements (z.B. das Edit-Steuerelement einer ComboBox). Das Steuerelement muss vom aktuellen Skript via Gui.Add erstellt worden sein.

Rückgabewert

Typ: Objekt oder Zeichenkette (leer)

Diese Funktion gibt das GuiControl-Objekt zurück, das mit dem angegebenen Fensterhandle verknüpft ist, oder eine leere Zeichenkette, wenn das zugehörige Objekt nicht gefunden wurde oder das Fensterhandle ungültig ist.

Bemerkungen

Das Fensterhandle eines GUI-Steuerelements kann z.B. mit GuiControl.Hwnd, MouseGetPos oder OnMessage abgerufen werden.
