MenuFromHandle

Ruft ein Menu- oder MenuBar-Objekt über dessen Win32-Menü-Handle ab.

Menu := MenuFromHandle(Handle)

Parameter

Handle

Typ: Integer

Der Handle eines Win32-Menüs (vom Typ HMENU).

Bemerkungen

Wenn das Handle ungültig ist oder nicht zu einem Menü gehört, das von diesem Skript erstellt wurde, gibt die Funktion eine leere Zeichenkette zurück.
