EnvSet

Schreibt einen Wert in eine bestimmte Umgebungsvariable.

EnvSet UmgebungVar , Wert

Parameter

UmgebungVar

Typ: Zeichenkette

Name der Umgebungsvariable, z.B. "Path".

Wert

Typ: Zeichenkette

Wenn weggelassen, wird die Umgebungsvariable gelöscht. Andernfalls geben Sie den zu schreibenden Wert an.

Fehlerbehandlung

Bei Misserfolg wird ein OSError ausgelöst.

Bemerkungen

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB Text.

Eine mit dieser Funktion erstellte oder geänderte Umgebungsvariable ist nur für Programme zugänglich, die im Skript via Run oder RunWait gestartet wurden. Weitere Informationen finden Sie unter Umgebungsvariablen.

Diese Funktion existiert, weil normale Skriptvariablen nicht in die Systemumgebung gespeichert werden. Das liegt zum einen daran, dass die Performanz darunter leiden würde, und zum anderen daran, dass das Betriebssystem die Größe jeder Umgebungsvariable auf 32 KB begrenzt.

Siehe auch

EnvGet, Run / RunWait

Beispiele

Schreibt etwas Text in eine Umgebungsvariable.

EnvSet "AutGUI", "Beliebiger Text für die Umgebungsvariable."
