ObjBindMethod

Erstellt ein BoundFunc-Objekt, das eine Methode des angegebenen Objekts aufruft.

BoundFunc := ObjBindMethod(Obj , Methode, Params)

Parameter

Obj

Typ: Objekt

Beliebiges Objekt.

Methode

Typ: Zeichenkette

Namen einer Methode. Wenn weggelassen, wird die Bound-Funktion Obj selbst aufrufen.

Params

Beliebig viele Parameter.

Bemerkungen

Informationen und Beispiele finden Sie unter BoundFunc-Objekt.
