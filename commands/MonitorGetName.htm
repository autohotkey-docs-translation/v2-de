MonitorGetName

Gibt den im Betriebssystem benutzten Namen eines bestimmten Bildschirms zurück.

Name := MonitorGetName(N)

Parameter

N

Typ: Integer

Eine Bildschirmnummer zwischen 1 und dem Rückgabewert von MonitorGetCount. Lässt man diesen Parameter weg, wird der primäre Bildschirm verwendet.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt den im Betriebssystem benutzten Namen des angegebenen Bildschirms zurück.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.
