Ord

Gibt den Ordinalwert (numerischen Zeichencode) des ersten Zeichens in einer bestimmten Zeichenkette zurück.

Nummer := Ord(Zeichenkette)

Parameter

Zeichenkette

Typ: Zeichenkette

Eine Zeichenkette, deren Ordinalwert abgerufen werden soll.

Rückgabewert

Typ: Integer

Diese Funktion gibt den Ordinalwert (numerischen Zeichencode) des ersten Zeichens in der angegebenen Zeichenkette zurück. Wenn die Zeichenkette leer ist, wird eine 0 zurückgegeben. Wenn die Zeichenkette mit einem Unicode-Ergänzungszeichen beginnt, wird diese Funktion den entsprechenden Unicode-Zeichencode zurückgeben (eine Nummer zwischen 0x10000 und 0x10FFFF). Ansonsten gibt sie einen Wert im Bereich von 0 bis 0xFF (bei ANSI) oder im Bereich von 0 bis 0xFFFF (bei Unicode) zurück. Einzelheiten finden Sie unter Unicode vs ANSI.
