GuiCtrlFromHwnd

Ermittelt das GuiControl-Objekt eines GUI-Steuerelements, das zu einer bestimmten HWND-Nummer gehört.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

Parameter

Hwnd

Typ: Integer

Die HWND-Nummer (eindeutige ID) eines GUI-Steuerelements. So ein Steuerelement könnte via Gui.Add erstellt worden sein.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu einem existierenden GuiControl-Objekt zurück. Dieses Objekt hat Methoden und Eigenschaften zum Modifizieren von GUI-Steuerelementen und Abrufen ihrer Informationen.

Bemerkungen

Zum Beispiel kann die HWND-Nummer (eindeutige ID) eines GUI-Steuerelements via GuiCtrl.Hwnd, MouseGetPos oder OnMessage abgerufen werden.
