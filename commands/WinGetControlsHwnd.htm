WinGetControlsHwnd

Gibt die eindeutigen ID-Nummern aller Steuerelemente in einem bestimmten Fenster zurück.

Steuerelemente := WinGetControlsHwnd(FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Array

Diese Funktion gibt ein Array zurück, das die HWND-Nummern (eindeutige IDs) aller Steuerelemente im angegebenen Fenster enthält. Wenn keine passenden Fenster gefunden werden, wird eine leere Zeichenkette zurückgegeben.

Wenn zum Beispiel der Rückgabewert in eine Variable namens Steuerelemente gespeichert wird und zwei Steuerelemente vorhanden sind, enthält Steuerelemente[1] die ID des ersten Steuerelements, Steuerelemente[2] die ID des zweiten Steuerelements und Steuerelemente.Length gibt die Zahl 2 zurück.

Steuerelemente werden nach ihrer Z-Reihenfolge sortiert, was üblicherweise der Navigation via Tab entspricht, sofern vom Fenster unterstützt.

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Die ID des Fensters oder Steuerelements, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
