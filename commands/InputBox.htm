InputBox

Zeigt ein Eingabefenster an, das den Benutzer zur Eingabe einer Zeichenkette auffordert.

Obj := InputBox(Anzeigetext, Titel, Optionen, Standardwert)

Parameter

Anzeigetext

Typ: Zeichenkette

Text des Eingabefensters, der den Benutzer darauf hinweist, was eingegeben werden soll. Wenn der Anzeigetext relativ lang ist, kann er via Fortsetzungsbereich in mehreren kurzen Zeilen aufgeteilt werden, um die Übersichtlichkeit und Wartbarkeit des Skripts zu erhöhen.

Titel

Typ: Zeichenkette

Der Titel des Eingabefensters. Lässt man diesen Parameter weg, wird standardmäßig der aktuelle Wert von A_ScriptName verwendet.

Optionen

Typ: Zeichenkette

Eine Zeichenkette von nicht-Groß-/Kleinschreibung-sensitiven Optionen, die mit Leer- oder Tabulatorzeichen voneinander getrennt sind:

Xn Yn: Die X- und Y-Koordinaten des Dialogfensters. Zum Beispiel würde X0 Y0 das Fenster in die obere linke Ecke des Desktops setzen. Lässt man eine dieser Koordinaten weg, wird das Fenster in dieser Abmessung zentriert. Jede Koordinate kann negativ sein, um das Dialogfenster teilweise oder vollständig über den Bildschirmrand hinaus zu positionieren.

Wn Hn: Die Breite und Höhe des Clientbereichs (das ist im Dialogfenster der Bereich ohne Titelleiste und Rahmen). Zum Beispiel W200 H100.

T: Bestimmt ein Zeitlimit in Sekunden. Zum Beispiel wäre T10.0 ein Zeitlimit von zehn Sekunden. Wenn ein Zeitlimit höher als 2147483 (24,8 Tage) angegeben ist, wird es automatisch auf 2147483 zurückgesetzt. Nachdem das Zeitlimit überschritten wurde, wird das InputBox-Fenster automatisch geschlossen und Result auf das Wort "Timeout" gesetzt. Value wird weiterhin alles enthalten, was der Benutzer zuletzt eingegeben hat.

Password: Ersetzt die Benutzereingabe mit Maskierungszeichen. Das Maskierungszeichen kann wie folgt geändert werden: Password*

Standardwert

Typ: Zeichenkette

Eine Zeichenkette, die zu Beginn im Eingabefeld von InputBox erscheint, sobald das Dialogfenster das erste Mal angezeigt wird. Der Benutzer kann die Zeichenkette z. B. durch Drücken der Rücktaste ändern.

Rückgabewert

Typ: Objekt

Diese Funktion gibt ein Objekt mit folgenden Eigenschaften zurück:

  • Value (Zeichenkette): Den vom Benutzer eingegebenen Text.
  • Result (Zeichenkette): Eines der folgenden Wörter, das angibt, wie das Eingabefenster geschlossen wurde: "OK", "Cancel", "Timeout".

Bemerkungen

Ein typisches Eingabefenster sieht wie folgt aus:

Das Dialogfenster erlaubt den Benutzer, einen beliebigen Text einzugeben und dann OK oder CANCEL (Abbrechen) zu drücken. Der Benutzer kann die Größe des Dialogfensters durch Ziehen des Rahmens ändern.

Ein GUI-Fenster kann ein modales InputBox-Fenster mithilfe der OwnDialogs-Option anzeigen. Ein modales InputBox-Fenster hindert den Benutzer daran, mit dem GUI-Fenster zu interagieren, bis er das InputBox-Fenster wieder schließt.

Beispiele

#1

Passwort := InputBox("(Ihre Eingabe wird versteckt)", "Bitte Passwort eingeben", "password").Value

#2

IB := InputBox("Bitte geben Sie eine Telefonnummer ein.", "Telefonnummer", "w640 h480")
if IB.Result = "Cancel"
    MsgBox "Sie haben '" IB.Value "' eingegeben, aber dann auf Abbrechen gedrückt."
else
    MsgBox "Sie haben '" IB.Value "' eingegeben."
