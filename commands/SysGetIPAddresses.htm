SysGetIPAddresses

Gibt ein Array mit den IPv4-Adressen des Systems zurück.

Adressen := SysGetIPAddresses()

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Typ: Array mit Zeichenketten

Diese Funktion gibt ein Array zurück, wo jedes Element eine IPv4-Adresse wie "192.168.0.1" ist.

Bemerkungen

Derzeit wird nur IPv4 unterstützt.

Diese Funktion gibt nur die IP-Adressen von den Netzwerkadaptern des Systems zurück. Die Funktion kann nicht die öffentliche (Internet-) IP-Adresse des Computers abrufen, wenn der Computer über einen Router mit dem Internet verbunden ist. Um die öffentliche IP-Adresse des Computers zu ermitteln, nutzen Sie eine externe Web-API. Zum Beispiel:

whr := ComObjCreate("WinHttp.WinHttpRequest.5.1")
whr.Open("GET", "https://api.ipify.org")
whr.Send()
MsgBox "Öffentliche IP-Adresse: " whr.ResponseText
