WinGetStyle / WinGetExStyle

Gibt den Style bzw. erweiterten Style eines bestimmten Fensters zurück.

Style := WinGetStyle(FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)
ExStyle := WinGetExStyle(FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Integer

Diese Funktionen geben entweder den Style oder erweiterten Style des angegebenen Fensters zurück. Wenn keine passenden Fenster gefunden werden, wird eine leere Zeichenkette zurückgegeben.

Fehlerbehandlung

Es wird eine Ausnahme ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Die Style-Tabelle beinhaltet eine Auflistung einiger Styles.

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

#1: Stellt fest, ob ein Fenster den WS_DISABLED-Style hat:

Style := WinGetStyle("Mein Fenstertitel")
if (Style & 0x8000000)  ; 0x8000000 ist WS_DISABLED.
    MsgBox "Mit dem Fenster kann nicht interagiert werden."

#2: Stellt fest, ob ein Fenster den WS_EX_TOPMOST-Style hat (immer im Vordergrund):

ExStyle := WinGetExStyle("Mein Fenstertitel")
if (ExStyle & 0x8)  ; 0x8 ist WS_EX_TOPMOST.
    MsgBox "Das Fenster ist immer im Vordergrund."
