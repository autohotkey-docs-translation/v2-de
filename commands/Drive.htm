Drive-Funktionen

Funktionen zum Abrufen diverser Informationen über die Laufwerke des Computers oder zum Durchführen von Änderungen an einem Laufwerk. Klicken Sie auf einen Funktionsnamen, um mehr zu erfahren.

Funktion Beschreibung
DriveEject Öffnet oder schließt ein bestimmtes CD- oder DVD-Laufwerk.
DriveGetCapacity Gibt die Gesamtkapazität des Laufwerks, das einen bestimmten Pfad enthält, in Megabyte zurück.
DriveGetFileSystem Gibt den Dateisystemtyp eines bestimmten Laufwerks zurück.
DriveGetLabel Gibt die Datenträgerbezeichnung eines bestimmten Laufwerks zurück.
DriveGetList Gibt eine Reihe von Buchstaben zurück, die jeweils einen Laufwerksbuchstaben im System repräsentieren.
DriveGetSerial Gibt die Datenträgerseriennummer eines bestimmten Laufwerks zurück.
DriveGetSpaceFree Gibt den freien Speicherplatz des Laufwerks, das einen bestimmten Pfad enthält, in Megabyte zurück.
DriveGetStatus Gibt den Status des Laufwerks, das einen bestimmten Pfad enthält, zurück.
DriveGetStatusCD Gibt den Medienstatus eines bestimmten CD- oder DVD-Laufwerks zurück.
DriveGetType Gibt den Typ des Laufwerks, das einen bestimmten Pfad enthält, zurück.
DriveLock Verhindert, dass ein bestimmtes Laufwerk ausgefahren werden kann.
DriveSetLabel Ändert die Datenträgerbezeichnung eines bestimmten Laufwerks.
DriveUnlock Bewirkt, dass ein bestimmtes Laufwerk wieder ausgefahren werden kann.

Fehlerbehandlung

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt. Zusätzlich wird, sofern nicht anders angegeben, jede Funktion bei Erfolg eine 1 (True) und bei Misserfolg eine 0 (False) zurückgeben.

Beispiele

#1: Erlaubt dem Benutzer, ein Laufwerk zum Analysieren auszuwählen:

Ordner := DirSelect( , 3, "Laufwerk zum Überprüfen auswählen:")
if not Ordner
    return
MsgBox
(
    "Alle Laufwerke: " DriveGetList() "
    Ausgewähltes Laufwerk: " Ordner "
    Typ: " DriveGetType(Ordner) "
    Status: " DriveGetStatus(Ordner) "
    Kapazität: " DriveGetCapacity(Ordner) " MB
    Freier Speicher: " DriveGetSpaceFree(Ordner) " MB
    Dateisystem: " DriveGetFilesystem(Ordner) "
    Bezeichnung: " DriveGetLabel(Ordner) "
    Seriennummer: " DriveGetSerial(Ordner)
)
