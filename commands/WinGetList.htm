WinGetList

Gibt die eindeutigen ID-Nummern von allen vorhandenen Fenstern zurück, die bestimmte Kriterien erfüllen.

Liste := WinGetList(FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Array

Diese Funktion gibt ein Array zurück, das die HWND-Nummern (eindeutige IDs) von allen vorhandenen Fenstern enthält, die den angegebenen Parametern entsprechen. Wenn keine passenden Fenster gefunden werden, wird ein leeres Array zurückgegeben.

Wenn zum Beispiel der Rückgabewert in eine Variable namens Liste gespeichert wird und zwei Fenster vorhanden sind, enthält Liste[1] die ID des ersten Fensters, Liste[2] die ID des zweiten Fensters und Liste.Length gibt die Zahl 2 zurück.

Die Fenster werden von oben nach unten abgerufen (entsprechend der Tatsache, wie sie auf dem Desktop gestapelt sind).

Bemerkungen

Um alle Fenster des gesamten Systems abzurufen, müssen Sie alle vier Titel-/Text-Parameter weglassen.

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

Besucht alle Fenster des gesamten Systems und zeigt Informationen zu jedem an.

IDs := WinGetList(,, "Program Manager")
for diese_ID in IDs
{
    WinActivate diese_ID
    diese_Klasse := WinGetClass(diese_ID)
    dieser_Titel := WinGetTitle(diese_ID)
    Result := MsgBox(
    (
        "Gehe alle Fenster durch
        " A_Index " von " IDs.Length "
        ahk_id " diese_ID "
        ahk_class " diese_Klasse "
        " dieser_Titel "

        Weiter?"
    ),, 4)
    if (Result = "No")
        break
}
